# AI Programming Foundations Project

This project builds a reproducible data workflow using public Cincinnati Fire/EMS CAD incident data. The workflow loads, cleans, explores, visualizes, and summarizes real municipal incident data using Python, pandas, and Jupyter Notebook.

## Dataset

Dataset name: Cincinnati Fire Incidents (CAD) including EMS ALS/BLS

Source: https://data.cincinnati-oh.gov/Safety/Cincinnati-Fire-Incidents-CAD-including-EMS-ALS-BL/vnsz-a3wp

Local file used in this project:

cincinnati_fire_incidents_2025.csv

The dataset contains public fire and EMS incident response records from Cincinnati, Ohio. This project uses the data for academic workflow practice only and does not include patient data, departmental data, or PHI.

## Project Files

data_workflow.ipynb

Main notebook for loading, cleaning, EDA, visualization, and summary.

cincinnati_fire_incidents_2025.csv

Public dataset used by the notebook.

module_summary.pdf

Final project summary report.

requirements.txt

Python dependencies needed to reproduce the notebook.

README.md

Project overview and run instructions.

## How to Run

1\. Clone or download this repository.

2\. Open a terminal in the project folder.

3\. Install dependencies:

pip install -r requirements.txt

4\. Start Jupyter:

jupyter notebook

5\. Open and run:

data_workflow.ipynb

Run the notebook from top to bottom.

## Notes

This project is limited to data workflow practice. It does not perform machine learning, prediction, clinical decision support, or operational deployment.
